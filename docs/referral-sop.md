# Bloom Corner Kiddies — Referral Program SOP (C1.5)

This document outlines the manual operating procedure for running the "Bloom Corner Kiddies" referral program using the built-in Admin Panel tools.

---

## 1. The Strategy
- **Referrer Reward:** [X]% discount on their next order.
- **New Buyer Reward:** [X]% discount on their first order.
- **Structure:** Accumulative. (e.g., 3 successful referrals = 3 separate discount uses or one larger reward, depending on your current store policy).

---

## 2. Creating a New Referral Code
**When:** After a customer makes their first purchase.

1.  Log in to the **Admin Panel**.
2.  Navigate to **Referrals**.
3.  Click **"Generate New Code"**.
4.  **Referrer Name:** Enter the parent's name.
5.  **WhatsApp:** Enter their number (format: 234...).
6.  **Code Format:** Use the pattern `BCK-[NAME][ID]` (e.g., `BCK-SARAH042`).
7.  Click **Save**.
8.  Send the code to the parent using the **Post-Purchase Template**.

---

## 3. Redeeming a Code (New Buyer)
**Scenario:** A new parent messages you on WhatsApp saying "Hi, my friend Sarah referred me, her code is BCK-SARAH042".

1.  **Verify the Code:**
    - Go to **Admin → Referrals**.
    - Find the code in the list.
    - Check if it is **Active**.
2.  **Apply the Discount:**
    - Calculate the [X]% discount on their total.
    - Confirm the order with the discounted price.
3.  **Record the Usage (Crucial):**
    - In the Admin Panel, click on the **Referral Code** (e.g., BCK-SARAH042).
    - Click **"Record New Use"**.
    - Enter the **New Buyer's Name** and **WhatsApp Number**.
    - This increments the `Total Referrals` for the original referrer and creates a permanent record.

---

## 4. Rewarding the Referrer
**Scenario:** Sarah's code has been used. You now owe Sarah a discount.

1.  Check the **Referrals List** periodically to see who has new "Redemptions".
2.  **Notify the Referrer:**
    > "Hi Sarah! Just wanted to let you know your friend [New Buyer Name] just made their first purchase using your code! 🥳 
    > 
    > As a thank you, I’ve added a **[X]% discount** to your account for your next order. Just remind me when you're ready to shop!"
3.  When Sarah shops again, apply her reward and (optional) add a note in your CRM or the admin system if you want to track that the reward was "claimed".

---

## 5. Program Maintenance
- **Review Top Referrers:** Once a month, look at the "Total Referrals" column. Consider sending a surprise gift or a higher discount to your top 3 referrers to keep them motivated.
- **Deactivate Old Codes:** If a customer hasn't shopped in over a year, you can set their code to **Inactive** to keep the list clean.
